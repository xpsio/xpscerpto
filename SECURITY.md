# Security Policy — XPScerpto
**Last updated:** 2025-11-03

## Supported Versions
We follow **SemVer**. Security fixes target the latest minor within supported majors.

| Version | Supported |
|---------|-----------|
| 0.x     | ✅ (best effort) |
| < 0.x   | ❌ |

## Reporting a Vulnerability
Please email **security@xpsio.com** (replace with your security inbox).  
Include:
- A clear description, impact, and minimal PoC if possible.
- Affected versions/commit, OS/toolchain info.
- Whether the issue is exploitable remotely/locally.

We aim to acknowledge within **72 hours** and provide a remediation plan within **7 days** for critical issues.

> Optional: publish a PGP key fingerprint here for encrypted reports.

## Coordinated Disclosure
- We prefer **responsible disclosure**: please allow us to release a fix and advisory before publicizing.
- Credit will be attributed unless you request anonymity.
